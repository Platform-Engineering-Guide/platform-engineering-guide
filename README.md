# The Comprehensive Guide to Platform Engineering — 2026 Edition

Companion repository for [The Comprehensive Guide to Platform Engineering](https://platformengineeringguide.com) by Jordan Dinsdale.

550 pages. 32 chapters. A practical, opinionated guide to building and scaling internal developer platforms — written for engineering leaders who need to make
real decisions in 2026.

---

## What is in this repository

This repository contains all supporting materials that accompany the guide:

- **Examples** — Code samples and configurations from every chapter, organised by part
- **Templates** — Production-ready templates referenced in the guide including IaC, pipeline definitions, and golden path scaffolding
- **Diagrams** — Architecture diagrams from the guide in editable formats
- **Reference Materials** — Supplementary reference content and reading lists
- **Errata** — Corrections and clarifications to the published edition
