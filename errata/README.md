# The Comprehensive Guide to Platform Engineering — Companion Repository

This repository is the official errata and update log for  
**The Comprehensive Guide to Platform Engineering — 2026 Edition**  
by Jordan Dinsdale.

## What This Repository Is For

Platform engineering moves fast. This repository exists to keep the book
honest between editions by tracking:

- **Corrections** to factual errors, outdated tool references, and broken
  links discovered after publication
- **Watch This Space resolutions** — updates to sections flagged in the
  book as areas of rapid change
- **Edition changelogs** — a summary of what changed between editions for
  readers deciding whether to upgrade

This is not a discussion forum or a place to debate the book's opinions.
It is a maintenance log for a living reference.

## The Living Edition Philosophy

This book takes the position that a technical reference which avoids
specifics in order to stay evergreen is less useful than one that is
precise about the current state of the field and honest about where
things are moving. That approach has a cost: specific claims age. This
repository is how that cost is managed.

Sections marked **Watch This Space** in the book were identified at
time of writing as areas likely to change before the next edition.
The [watch-this-space.md](watch-this-space.md) file tracks how those
areas have evolved.
